# Default LAIRCA Ordering

A broad default ordering for LAIRCA should remain open to refinement, but a strong starting pattern is:

1. Theology proper and final ends
2. Anthropology and imago Dei
3. Ordered loves, humility, and sin
4. Vocation, stewardship, and mission
5. Justice, public faithfulness, and institutional responsibility
6. Prudence, power, and governance
7. Formation, service, and operator character
8. Tools, automation, and technical control

## Why this order
It preserves metaphysical and moral priority before operational and technical layers.
