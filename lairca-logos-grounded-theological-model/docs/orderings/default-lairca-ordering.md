# Default LAIRCA Ordering

## Purpose

This file proposes a default broad-Christian ordering profile for using the Logos project as an upstream architecture for LAIRCA.

## Design rule

This is an ordering profile, not a replacement for doctrine nodes.

## Suggested default order

1. Logos and truth
2. doctrine of God
3. Christology
4. revelation and Scripture
5. anthropology and human dignity
6. sin and fallenness
7. wisdom, prudence, and moral reasoning
8. justice, mercy, and neighbor-love
9. vocation, stewardship, and responsibility
10. authority, community, and institutional ordering
11. technology, tools, and mediation
12. application-specific optimization

## Why this order

This sequence keeps optimization downstream of theological truth, moral anthropology, and covenantal responsibility.

It prevents efficiency from becoming a hidden first principle.

It also ensures that AI-supported systems are interpreted through a prior understanding of personhood, agency, sin, prudence, and moral accountability.

## Practical implication

Any downstream system using this ordering should test whether:
- truth is being distorted
- persons are being reduced to metrics
- responsibility is being displaced
- efficiency is outranking moral order
- automation is crossing theological boundaries

## Status

Working default. Open to refinement as doctrine nodes and comparative overlays mature.
