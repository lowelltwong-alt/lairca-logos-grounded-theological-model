# AIRCA Companion Crosswalk

## Purpose

Explain how this repository connects to the AIRCA Fractal Decision Architecture project.

Companion repo:
- https://github.com/lowelltwong-alt/airca-fractal-decision-architecture

## Division of responsibility

### AIRCA repo
- AIRCA operational architecture
- UAIRCA abstract U-layer logic
- concise LAIRCA framing
- cross-framework architecture

### This repo
- Christian theological source architecture
- canon and doctrine nodes
- ordering and weighting logic
- theology-to-LAIRCA derivation chains
- optional tradition overlays
- Habitat and other application derivations

## Key rule

The theological model does not replace AIRCA. It configures one Christian instantiation of the U-layer that AIRCA can operate under.

## Connection points to preserve
- mirrored root files
- repeated node structure
- compatible artifact schema style
- explicit `related_to` and `derived_from` references
- same author attribution convention
