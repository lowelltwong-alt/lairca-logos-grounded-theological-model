# Default AI Governance Weighting

## Purpose

This file provides a starter weighting profile for AI governance use cases derived from the Logos project.

## Principle

Weighting is not the same as ordering.

Ordering describes what should be considered in sequence.
Weighting describes the relative force a consideration should carry in a given use case.

## Starter weighting profile

### Very high weight
- truthfulness
- imago Dei and human dignity
- human responsibility and answerability
- protection against deception and coercion
- moral limits on means

### High weight
- prudence
- justice
- stewardship
- transparency of lineage and governance
- preservation of human discernment in consequential contexts

### Moderate weight
- institutional efficiency
- consistency
- speed
- scalability
- automation of low-consequence routine work

### Conditional weight
- personalization
- optimization against local KPIs
- autonomous action without review
- black-box ranking in morally significant contexts

## Practical test

A proposed AI-enabled workflow should fail review when low-level efficiency gains are purchased by violating higher-weight commitments.

## Use

This profile is designed for early-stage Christian AI ethics, nonprofit governance, and decision-support applications.
