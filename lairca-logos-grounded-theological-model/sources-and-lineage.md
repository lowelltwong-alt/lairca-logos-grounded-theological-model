## What is inherited conceptually

Inherited from the AIRCA-side ecosystem:
- fractal artifact logic
- recursive node pattern
- machine-readable sidecar discipline
- lifecycle and governance gating
- explicit lineage and derivation handling
- LAIRCA as a Christian instantiation of the U-layer

## What is original here

This repository adds:
- broad orthodox Christian source canon
- doctrine ordering profiles
- weighting logic for Christian institutional discernment
- derivation chains from theology into governance and decision architecture
- optional tradition overlays
- Habitat-facing and other institutional application branches

This repository is designed to prevent "Christian framing" from remaining vague. It makes the theological substrate inspectable, orderable, weightable, and machine-readable.
